═══════════════════════════════════════════════════════
  HOW TO HOST (choose ONE option)
═══════════════════════════════════════════════════════

OPTION 1 — GitHub Pages (FREE, Recommended)
────────────────────────────────────────────
1. Go to github.com → New repository → Name: "red-nectar"
2. Upload all 5 files (index.html, manifest.json, sw.js, icon-192.png, icon-512.png)
3. Go to Settings → Pages → Source: "main branch" → Save
4. Your app is live at: https://YOUR_USERNAME.github.io/red-nectar/

OPTION 2 — Netlify (FREE, Drag & Drop)
────────────────────────────────────────
1. Go to netlify.com → Sign up free
2. Drag the entire rednectar-pwa FOLDER onto the Netlify dashboard
3. Done! You get a URL like: https://rednectar-xyz.netlify.app

OPTION 3 — Firebase Hosting (FREE)
────────────────────────────────────
1. Go to console.firebase.google.com
2. New project → Hosting → Deploy
3. firebase init hosting → firebase deploy

═══════════════════════════════════════════════════════
  INSTALL AS APP ON PHONE
═══════════════════════════════════════════════════════

Android (Chrome):
  1. Open the hosted URL in Chrome
  2. Tap the "⋮" menu → "Add to Home Screen"
  3. Tap Install — app icon appears on home screen!

iPhone (Safari):
  1. Open the hosted URL in Safari
  2. Tap the Share button (□↑)
  3. Tap "Add to Home Screen" → Add
